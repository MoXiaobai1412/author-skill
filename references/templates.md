# 小说创作模板

## list.md 索引格式（每个库文件夹下）

```markdown
# {库名}索引

| 资源名 | 资源概况 | 相关资源 |
|--------|----------|----------|
| 林远 | 主角，少年，坚毅 | 清风（救命恩人）、林父（父亲） |
| 清风 | 散修，路过救下林远 | 林远（救命恩人） |
| ... | ... | ... |

---
*使用相应命令添加资源后，此处会自动更新*
```

**说明：**
- 每个库文件夹下必须有 `list.md` 作为索引
- 资源名：人物名/设定名/伏笔 ID 等
- 资源概况：简要描述（50 字以内）
- 相关资源：关联的人物/设定/伏笔

## ✅ 标准列名（所有库必须包含）

每个资源库的 list.md **必须**包含以下 4 列：

| 列名 | 英文名 | 类型 | 必填 | 说明 |
|------|--------|------|------|------|
| **资源名** | name | text | YES | 资源名称（重要资源加⭐） |
| **资源概况** | summary | text | YES | 资源的简要概述 |
| **相关资源** | relations | text | NO | 关联的其他资源 |
| **出现章节** | chapter | text | NO | **所有**出现章节的列表 |

## 🔧 更新出现章节的方法

### 写作后必须更新

**每次完成章节后：**

1. **检测新资源**
   - 本章出场的新人物
   - 本章涉及的新设定
   - 本章铺设/回收的伏笔

2. **更新 chapter 列**
   ```
   原：第 1,2,3 章
   本章：第 5 章
   新：第 1,2,3,5 章
   ```

3. **更新 relations（如有新关系）**
   ```
   原：清风（救命恩人）
   本章：与林父相认
   新：清风（救命恩人）、林父（父亲）
   ```

### 自动化脚本

```bash
# 写作后自动更新
python scripts/update_resources.py books/{书名} --chapter 第 X 章

# 手动更新
python scripts/resource_manager.py update_chapter {书名} books 人物库 林远 "第 1,2,3,5 章"
```

## 📊 list.md 长度管理

### 长度限制建议

| 库类型 | 建议最大行数 | 超限处理 |
|--------|------------|---------|
| 人物库 | 200 行 | 分类：主角团、配角团、反派团 |
| 设定库 | 300 行 | 分类：世界观、功法、物品、组织 |
| 场景库 | 150 行 | 分类：东域、西域、南域等 |
| 伏笔库 | 100 行 | 分类：主线伏笔、支线伏笔 |
| 历史库 | 100 行 | 分类：上古、近代、现代 |
| 人物关系库 | 200 行 | 分类：血缘、师徒、仇敌等 |

### 分页策略

**当 list.md > 500 行时：**

1. **分类管理**
   ```
   人物库/
   ├── list.md（总索引，只列分类）
   ├── 主角团/
   │   └── list.md
   ├── 配角团/
   │   └── list.md
   └── 反派团/
       └── list.md
   ```

2. **重要标记**
   - ⭐ 重要资源（贯穿全文）
   - 普通资源（支线相关）

3. **归档机制**
   ```
   已完成卷的资源 → 归档库/
   当前卷资源 → 活跃库/
   ```

### AI 读取优化

**LLM 检索策略：**

```
1. 读 desc（了解结构）- 10 行
   ↓
2. 筛选⭐重要资源 - 10-20 行
   ↓
3. 关键词匹配相关条目 - 10-30 行
   ↓
4. 读取具体档案文件 - 按需

总计：30-60 行，精准高效
```

**避免：**
- ❌ 每次读取整个 list.md（1000+ 行）
- ❌ 不筛选直接加载
- ❌ 忽略⭐标记

**推荐：**
- ✅ 先读 desc
- ✅ 优先⭐重要资源
- ✅ 关键词检索相关条目
- ✅ 按需读取档案文件

## ✅ 检查清单

每次更新资源库时检查：

- [ ] **name 列** - 资源名（重要资源加⭐）
- [ ] **summary 列** - 资源概况清晰
- [ ] **relations 列** - 相关资源完整
- [ ] **chapter 列** - **所有**出现章节（不只是首次）
- [ ] **chapter 格式** - 使用 `第 1,2,3 章` 或 `第 1-5 章`
- [ ] **list.md 长度** - 未超过建议上限
- [ ] **分类管理** - 超限时已分类
- [ ] **归档处理** - 已完成卷已归档

## 📝 示例：完整更新流程

### 场景：完成第 5 章

**1. 检测新资源**
```
本章出场：清风（第 2 章已出场，非新）
本章新设定：青云宗试炼塔（新）
本章伏笔：试炼塔顶层的神秘声音（新伏笔）
```

**2. 添加新资源**
```bash
# 添加新设定
python scripts/resource_manager.py add 未来救援局 books 设定库 试炼塔 "青云宗试炼场地，共 9 层" "青云宗、试炼规则" "第 5 章"

# 添加新伏笔
python scripts/resource_manager.py add 未来救援局 books 伏笔库 神秘声音 "试炼塔顶层传来的声音" "试炼塔、未知存在" "第 5 章"
```

**3. 更新已有资源**
```bash
# 更新林远的出现章节
python scripts/resource_manager.py update_chapter 未来救援局 books 人物库 林远 "第 1,2,3,5 章"

# 更新清风的出现章节
python scripts/resource_manager.py update_chapter 未来救援局 books 人物库 清风 "第 1,2,5 章"
```

**4. 验证**
```bash
# 查看人物库
python scripts/resource_manager.py list 未来救援局 books 人物库

# 输出：
| 资源名 | 资源概况 | 相关资源 | 出现章节 |
|--------|----------|----------|----------|
| 林远 ⭐ | 主角，少年，坚毅 | 清风、林父 | 第 1,2,3,5 章 |
| 清风 | 散修，救下林远 | 林远 | 第 1,2,5 章 |
```

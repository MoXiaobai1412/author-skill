# 快速上手指南

## 首次使用

### 1. 创建新书
```
/newbook {书名}
```
示例：`/newbook 仙途`

**默认位置**：`workspace/books/{书名}/`

**自定义位置**：`/newbook 仙途 D:/my/novel/path`

系统会自动创建：
- **资源库/** 含 6 个子库（人物/设定/伏笔/高光/历史/自定义），每个都有 list.md
- **正文/** 正文存储
- **大纲/** 含总纲和卷纲，每个文件夹都有 list.md
- **概括/** 卷概括和章概括，每个文件夹都有 list.md
- **写作要求/** 文笔、对话风格等模板

### 2. 设置总纲
```
/outline add 第一卷：入道 "主角踏上修仙之路，初入宗门"
/outline chapter add 第一卷：入道 第一章：灭门 "林府被灭，林远逃生"
```

### 3. 添加主要人物
```
/res add 人物库 林远 "主角，少年，坚毅，林府灭门唯一幸存者" "清风（救命恩人）"
/res add 人物库 清风 "散修，路过救下林远，身份神秘" "林远"
```

### 4. 设置写作要求
```
/req edit 文笔.md    # 编辑文笔要求
/req edit 对话风格.md  # 编辑对话风格
```

### 5. 开始写作
```
/write 第一卷：入道 第一章：灭门
```

## 常用命令速查

### 书籍管理
| 命令 | 说明 |
|------|------|
| `/newbook {书名}` | 创建新书 |
| `/select {书名}` | 切换当前书 |
| `/listbooks` | 列出所有书 |

### 资源管理（核心）
| 命令 | 说明 |
|------|------|
| `/res add {库名} {资源名} [概况] [相关资源]` | 添加资源 |
| `/res edit {库名} {资源名}` | 编辑资源 |
| `/res del {库名} {资源名}` | 删除资源 |
| `/res list {库名}` | 列出库中所有资源 |
| `/res get {库名} {资源名}` | 获取资源详情 |
| `/res search {库名} {关键词}` | 搜索资源 |

**支持库名**：人物库、设定库、伏笔库、高光库、历史库、用户自定义库

### 伏笔专用
| 命令 | 说明 |
|------|------|
| `/vow update {ID} {状态} [实际揭露]` | 更新伏笔状态 |
| `/vow list` | 列出所有伏笔 |

### 大纲管理
| 命令 | 说明 |
|------|------|
| `/outline add {卷名} {概括}` | 添加卷纲 |
| `/outline chapter add {卷名} {章名} {概括}` | 添加章纲 |
| `/outline list [路径]` | 列出大纲索引 |
| `/outline get {卷名} [章名]` | 获取大纲内容 |

### 写作
| 命令 | 说明 |
|------|------|
| `/write {卷名} {章名}` | 写指定章节 |
| `/edit {卷名} {章名}` | 修改指定章节 |
| `/status` | 查看进度 |
| `/check` | 冲突检测 |
| `/export` | 导出正文 |

### 索引管理
| 命令 | 说明 |
|------|------|
| `/index read {路径}` | 读取 list.md |
| `/index add {路径} {名称} {概括} {状态}` | 添加索引条目 |
| `/index del {路径} {名称}` | 删除索引条目 |
| `/index sync {路径}` | 同步索引与文件 |
| `/index search {路径} {关键词}` | 搜索索引 |

**路径示例**：资源库、资源库/人物库、正文、大纲、大纲/第一卷、概括、写作要求

### 写作要求
| 命令 | 说明 |
|------|------|
| `/req add {文件名} [标题]` | 添加写作要求 |
| `/req edit {文件名}` | 编辑写作要求 |
| `/req del {文件名}` | 删除写作要求 |
| `/req list` | 列出所有要求 |
| `/req get {文件名}` | 获取要求详情 |

## 资源管理示例

### 添加人物
```bash
/res add 人物库 林远 "主角，少年，坚毅，林府灭门唯一幸存者" "清风（救命恩人）、林父（父亲）"
```

**效果：**
- 创建 `资源库/人物库/林远.md`
- 自动更新 `资源库/人物库/list.md`

### 查看人物列表
```bash
/res list 人物库
```

**输出：**
```
## 人物库资源列表（《仙途》）

| 资源名 | 概况 | 相关资源 |
|--------|------|----------|
| 林远 | 主角，少年，坚毅 | 清风、林父 |
| 清风 | 散修，路过救下林远 | 林远 |

共 2 个资源
```

### 搜索资源
```bash
/res search 人物库 林
```

### 添加设定
```bash
/res add 设定库 青云宗 "主角所在宗门，正道大派，历史悠久" "修仙体系、林远（弟子）"
```

### 添加伏笔
```bash
/res add 伏笔库 飘荡的窗帘 "诡异现象，暗示有鬼魂存在" "第一卷第 3 章"
```

**效果：**
- 自动分配伏笔 ID（如 V001）
- 创建 `资源库/伏笔库/飘荡的窗帘.md`
- 更新 `资源库/伏笔库/list.md`
- 更新 `资源库/伏笔库/伏笔列表.md`

### 更新伏笔状态
```bash
/vow update V001 已揭露 "第二卷第 5 章"
```

### 添加高光场景
```bash
/res add 高光库 五台山大战 "主角越级挑战，一战成名" "第一卷第 10 章"
```

### 添加历史事件
```bash
/res add 历史库 第一次企业战争 "50 年前的全球战争，改变世界格局" "背景设定"
```

## 写作流程

### 1. 写作前查阅
```bash
# 查看资源库有哪些子库
/index read 资源库

# 查看人物库，了解已有角色
/res list 人物库

# 查看某人物详情
/res get 人物库 林远

# 查看写作要求
/req list
/req get 文笔.md
```

### 2. 开始写作
```bash
/write 第一卷：入道 第一章：灭门
```

### 3. 写作完成后
- 自动生成章概括
- 更新概括/第一卷/章概括/list.md
- 更新进度
- Git 提交

## 核心原则

1. **list.md 是唯一索引**：所有操作先查 list.md
2. **先概况后详情**：通过 list.md 了解概况再调文件
3. **自动同步**：增删改资源时自动更新 list.md
4. **按需读取**：只加载需要的资源
5. **用户可扩展**：支持自定义库和写作要求
